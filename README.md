## 構成

- TypeScript
- Webpack
- TSLint
- Prettier

Lintのテンプレートは `tslint-config-standard` を利用しています。  
jQueryの型定義ファイルをインストール済みです。

## インストール

```
$ npm install
```

jsファイルのエントリーポイントを増やす場合は `webpack.config.js` を編集してください。

## 開発モード

```
$ npm start
```

## ビルド

```
$ npm run build
```

## 構文チェック&自動整形

```
$ npm run lint
```
